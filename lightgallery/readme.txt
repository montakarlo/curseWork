== Description ==

= Create beautiful image galleries for your WordPress website. =


This is an easy to use WordPress plugin of popular [lightGallery](http://sachinchoolur.github.io/lightGallery/) jQuery plugin.

= Main Features =

* Fully responsive
* Modular architecture
* Multiple galleries on one page
* Touch support for mobile devices
* Mouse drag supports for desktops
* 20+ Hardware-Accelerated CSS3 transitions
* Dynamic mode
* Easily customizable via CSS (SCSS) and Settings
* Smart image preloading and code optimization
* Keyboard Navigation for desktop
* Font icon support

= Usage =

1. Go to Post > Edit > LightGallery Meta Box
2. Select type of media you want to insert in your content
3. Required ShortCode will be generated automatically
4. Copy/Paste or use Insert button to add it to your content

== Installation ==

Search for the plugin name in Plugins menu > Add new and then click install or you can download and extract the plugin package and copy the plugin folder to your wp-content/plugins directory.
